Detailed Description

This class represents a virtualized cartridge parser.

The class capable to read and parse the NES2.0 (.nes) file format, which is also backward compatible with the older iNes (NES1.0) file format.

uint32_t Emulator::Cartridge::calcCHRRomSize ( )
private

calculates the character rom size.

Based on the rawHeader data calculates the program rom size. Using the rawHeader 5th byte as LSB and bits 4-7 of rawHeader 9th byte as MSB (12 bits in total). If the 4 MSB bits are 1111 then using the exponent-multiplier notation therefore size = 1111 EEEE EEMM |||| ||++- Multiplier, actual value is MM*2+1 (1,3,5,7) ++++-++— Exponent (2^E), 0-63 else: BBBB BBBB BBBB ++++-++++-++++- PRG-ROM size in 8 KiB units, values $000-$EFF for 0..31,449,088 bytes

Returns
the actual size of the chrRom in bytes.

uint32_t Emulator::Cartridge::calcPRGRomSize ( )
private

calculates the program rom size.

Based on the rawHeader data calculates the program rom size. Using the rawHeader 4th byte as LSB and bits 0-3 of rawHeader 9th byte as MSB (12 bits in total). If the 4 MSB bits are 1111 then using the exponent-multiplier notation therefore size = 1111 EEEE EEMM |||| ||++- Multiplier, actual value is MM*2+1 (1,3,5,7) ++++-++— Exponent (2^E), 0-63 else: BBBB BBBB BBBB ++++-++++-++++- PRG-ROM size in 16 KiB units, values $000-$EFF for 0..62,898,176 bytes

Returns
the actual size of the prgRom in bytes.
