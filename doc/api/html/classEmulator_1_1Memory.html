Detailed Description

NES Memory module representation.

This class represents the NES' Memory module including it's mirroring properties based on the specification.

◆ generateRAM()

void Emulator::Memory::generateRAM ( )
private

Implementing the RAM section mirroring.

RAM section mirroring Zero-page, Stack, RAM sections after each other (0x0000 - 0x07FF) mirrored in the Memory (0x8000 - 0x2000) 3 times. In total each section is presented in the Memory 4 times.

Emulator::Memory::memory

Represents the physical NES memory module.

It is a 0x10000 long 8bit unsinged integer pointer array. The size defined by the actual NES memory module meanwhile the pointer aspect was used to make the mirroring implementation easier.
